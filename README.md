 GearRentPro

Java-based Equipment Rental Management System using MySQL.

Project Description
GearRentPro is a desktop-based Java application developed to manage an equipment rental business. The system allows the administrator to manage categories, equipment, customers, reservations, and rentals using a centralized database.

 Technologies Used
- Java (Swing / JDBC)
- MySQL
- NetBeans IDE

 How to Configure the Database
1. Open MySQL Workbench.
2. Create a database named `gearrentdb`.
3. Open the file `gearrentdb_FINAL.sql`.
4. Execute the script to create tables and insert sample data.
5. Update database credentials in the DBConnection class if required.

 How to Run the Application
1. Open the project in NetBeans IDE.
2. Ensure the MySQL server is running.
3. Build the project.
4. Run the main class to start the application.
